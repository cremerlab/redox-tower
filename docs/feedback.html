Feedback

We welcome feedback on the interactive redox tower — whether it's a bug, a suggestion for a new metabolism or environmental preset, a question about the calculations, or anything else.

Open a GitHub issue

The preferred way to report bugs or suggest improvements is via the GitHub issue tracker. This keeps feedback visible and lets others contribute to the discussion.

Contact the authors

For questions about the science or the associated paper, you can reach Jonas Cremer or Orkun Soyer via their lab websites.
